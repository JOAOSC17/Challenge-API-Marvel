# App API da Marvel
Uma aplicação em que é possível procurar por heróis, quadrinhos e criadores de quadrinhos da Marvel.
## 🚀 Tecnologias Usadas:
- HTML
- CSS 
- JavaScript
## ℹ️ Como usar:
~~~ 
# Baixe o zip desse repositório.
# Entre no arquivo login.html
# E use as entradas padrão de email e senha:
- admin@admin.com
- Senha: admin
